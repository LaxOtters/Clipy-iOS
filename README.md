# Clipy iOS

웹 탐색 중 비교하고 싶은 항목을 세션으로 모아두고, 나중에 다시 돌아와 결정할 수 있게 돕는 iOS 앱입니다.

Clipy는 “저장”보다 “결정까지 이어지는 흐름”에 집중합니다.
사용자가 보던 맥락, 후보, 선택 상태를 잃지 않는 경험을 목표로 합니다.

## Focus

- 탐색 중 만난 항목을 세션 단위로 유지합니다.
- 비교와 결정을 위한 상태를 앱 안에 남깁니다.
- 재진입했을 때 이전 흐름을 자연스럽게 이어갑니다.

## iOS Direction

- Tuist 기반으로 프로젝트를 관리합니다.
- UI는 UIKit을 기준으로 가져갑니다.
- 필요한 만큼만 모듈을 나눕니다.
- 테스트는 핵심 비즈니스 규칙을 설명하는 문서처럼 남깁니다.
- 구조보다 제품 흐름이 먼저 읽히는 코드를 지향합니다.

개발 환경, 프로젝트 구조, 코드 기준은 `Docs/`에서 관리합니다.
